# Sisters Pages React

一个基于 Next.js、React 和 Tailwind CSS 构建的角色页面示例，支持多语言、地图、画廊等功能。

## 技术栈

- Next.js 15.x
- React 19.x
- TypeScript
- Tailwind CSS
- Radix UI
- Vercel

## 主要功能

- 响应式布局
- 主题切换（暗色/亮色模式）
- i18n 多语言支持
- 图片画廊、照片集
- 沉浸式地图
- 角色卡片展示

## 本地运行

```bash
pnpm install
pnpm dev
```

访问 http://localhost:3000

## 构建与部署

```bash
pnpm build
pnpm start
```
